# Quentin Tarantino Filmography

A web-based application built as a dynamic Java web project to display a curated list of films directed by Quentin Tarantino.

## Features
* **Dynamic Data Rendering:** Uses Java scriptlets to manage and iterate through movie data.
* **Responsive Design:** Styled with a custom, high-contrast dark theme inspired by crime-thriller aesthetics.
* **Data Fields:** Lists the Movie Title, Release Year, and Studio/Distributor for each entry.

## Requirements
* **Java Development Kit (JDK):** Version 8 or higher.
* **Server:** Apache Tomcat (or equivalent servlet container).
* **IDE:** Eclipse IDE for Enterprise Java and Web Developers.

## How to Run
1. Clone or import this project into your IDE.
2. Ensure your **Targeted Runtime** is set to Apache Tomcat.
3. Right-click the project, select **Run As** > **Run on Server**.
4. Access the project in your browser at `http://localhost:8080/[YourProjectName]/`.
